Added Parameter 'AutoSelectOnly' to Get-WUInstall from http://gallery.technet.microsoft.com/scriptcenter/2d191bcd-3308-4edd-9de2-88dff796b0bc 

This parameter filters updates based on the 'AutoSelectOnWebsites' property. Basically, this installs updates that are marked 'important' in the Windows Update GUI (as opposed to 'optional').

To use: Get-WUInstall -AutoSelectOnly -MicrosoftUpdate
