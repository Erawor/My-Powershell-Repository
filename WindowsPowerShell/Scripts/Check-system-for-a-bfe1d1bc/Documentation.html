You can use this scrit to identify if a special patch is installed on a system listed in the servers.txt file. you have to create a text file with the server names to be checked.
 
Insert the server name as shown below, followed by a CR/LF at the end and save it as servers.txt in the folder where the script is located.
servers.txt
server1
server2
server3
serverN
PowerShell
Edit|Remove
powershell 
$servers = Get-Content servers.txt
$KBPatch = 'KB2858725'
#$KBPatch = 'KB2978092'

#write-host $KBPatch $LogFile $Installed

Foreach ($server in $servers)
{
	if (!(get-hotfix -id $KBPatch -computername $server -ErrorAction 0))	
		{ 
		write-host -ForegroundColor "green" "$server - $KBPatch - not installed"
		} Else {
		write-host -ForegroundColor "red" "$server - $KBPatch - installed"
		}
}
$servers = Get-Content servers.txt 
$KBPatch = 'KB2858725' 
#write-host $KBPatch $LogFile $Installed 
 
Foreach ($server in $servers) 
{ 
    if (!(get-hotfix -id $KBPatch -computername $server -ErrorAction 0))     
        {  
        write-host -ForegroundColor "green" "$server - $KBPatch - not installed" 
        } Else { 
        write-host -ForegroundColor "red" "$server - $KBPatch - installed" 
        } 
}
 Change the value of $KBPatch to an Microsoft Knowledge Base article in the format "KB<ARTICLE_NUMBER>".
