Allows a much simpler management for all Microsoft Updates, which are registered in the wsusscn2.cab and their download from the official Microsoft Update sites. The Microsoft Updates Downloader PowerShell Module is e.g. perfect for patching of WIM (Windows Imaging File) files or Office 2013, 2010 and 2007 Installations. PowerShell 3.0 is requirement!

PowerShell
Skript bearbeiten|Remove
powershell 
Import-Module .\MS-UPD-LOAD.psm1

Get-UpdXML -Year "2013" -Month "03"

Get-Update -ProductFamily "Office"

Get-Update -ProductFamily "Windows"

Invoke-Update -Product "Office 2010" -Language English

Invoke-Update -Product "Windows Server 2012 R2"  -Language English

Get-UpdCommand
