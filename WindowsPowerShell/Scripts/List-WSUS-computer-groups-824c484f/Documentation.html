This simple script lists all computer groups defined on the local WSUS server displaying name and number of members in each group.
 
PowerShell
Edit|Remove
powershell 
[reflection.assembly]::LoadWithPartialName("Microsoft.UpdateServices.Administration")

$wsus = [Microsoft.UpdateServices.Administration.AdminProxy]::getUpdateServer()

foreach( $group in $wsus.GetComputerTargetGroups() )
{
    Write-Host "Group: " $group.Name " - " $group.GetComputerTargets().Count " member(s)"
}
